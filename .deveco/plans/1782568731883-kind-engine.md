## 功能清单（对齐 Android 原版）

### BookSourcePage
- **搜索**: TextInput 实时过滤
- **排序**: 7 种（手动/名称/URL/权重/更新时间/响应时间/启用）
- **分组标签**: 水平滚动 Chip 列表（全部/已启用/已禁用/自定义分组）
- **域名分组**: Toggle 切换，按域名分组显示
- **列表项**: 名称 + URL + 分组标签 + 权重 + 启用开关 + 探索标记点
- **批量操作**: 长按进入选择模式，底部操作栏（启用/禁用/分组/导出/校验/删除）
- **导入入口**: 顶部按钮 → ImportSourceDialog
- **导出**: 底部导出按钮 → DocumentViewPicker save
- **校验**: 调用 SourceChecker，显示进度条
- **分组管理**: 按钮 → GroupManageDialog

### ImportSourceDialog
- **预览列表**: 解析后显示所有书源，标记新增(绿)/更新(橙)/已有(灰)
- **勾选**: 每行 Checkbox，默认选中新增+更新
- **快捷操作**: 全选/仅选新增/仅选更新
- **选项**: 保留原名/保留分组/保留启用（Toggle）
- **自定义分组**: 输入框 + 追加/覆盖切换
- **底部**: 导入确认按钮 + 计数

### 导出
- 选中书源序列化为 JSON
- DocumentViewPicker save 写入文件
